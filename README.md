# Island Splash Creative Studio

## Project Overview

**Name:** Island Splash Creative Studio  
**Type:** Social Media Content Generator Web App  
**Purpose:** A platform for creating branded social media creatives (images/videos) for the Island Splash beverage brand

---

## Core Features

### 1. Brand Manager
- Add/manage brand logos, colors, fonts
- Save product images
- Brand profile management

### 2. Creative Generator
- Text-to-image generation
- Reference image upload + prompt
- Style templates (tropical, minimal, bold, etc.)
- AI-powered prompt enhancement

### 3. Template Library
- Pre-made layouts for posts, stories, reels
- Platform-specific sizes (9:16, 1:1, 16:9)
- Editable text overlays
- Background image library

### 4. Asset Library
- Upload/store product shots, logos
- Background image library
- Asset organization by category

### 5. Export
- Download in multiple formats (PNG, JPG, MP4)
- Direct export to social platforms
- Batch export support
