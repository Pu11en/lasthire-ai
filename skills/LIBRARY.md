# Lasthire.ai Library

A meta-skill for managing Lasthire.ai Pinterest affiliate marketing operations.

## What This Is

The Library is your command center for running automated tasks on Lasthire.ai. It manages skills that handle different parts of your affiliate business.

## Skills Available

### 🔍 Research
- `research-products` - Find high-ticket home decor products
- `analyze-trends` - Research trending Pinterest keywords
- `competitor-analysis` - See what competitors are pinning

### 🎨 Content
- `generate-pin-ideas` - Brainstorm pin concepts
- `write-descriptions` - SEO-optimized pin descriptions
- `create-captions` - Pinterest captions with keywords

### 📊 Analytics
- `track-performance` - Monitor clicks, saves, earnings
- `identify-winners` - Find top-performing pins
- `optimize-strategy` - Data-driven recommendations

### 🔗 Affiliate
- `apply-programs` - Auto-apply to affiliate programs
- `manage-links` - Track and organize affiliate links
- `test-links` - Verify links are working

### 🚀 Automation
- `schedule-pins` - Plan content calendar
- `batch-create` - Create multiple pins at once
- `report-weekly` - Generate weekly performance reports

### 🛠️ Vercel
- `debug` - Check deployment status, logs, redeploy

## How To Use

Tell me what you want to do and I'll use the appropriate skill:

- "Find me products to promote" → runs research-products
- "What pins are working?" → runs track-performance
- "Apply to more affiliate programs" → runs apply-programs

## Manual First

Always test manually before automating:
1. Do it yourself first
2. Note what works
3. Then automate
